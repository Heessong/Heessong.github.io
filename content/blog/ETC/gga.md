요즘 사람들이 많이 쓰고 있는 다양한 앱들과 웹서비스들은 이용자들의 데이터를 수집하여 이용자가 관심을 가질만한 내용들을 계속 화면에 추천해주거나 팝업메세지, 메일 등 지속적으로 이용자에게 제안하여 본인들의 서비스에 더 오래 머물수있도록 유도한다. 어느기업이든 다 사용자의 정보를 기반으로 마케팅과 비즈니스를 하지만 특히나 구글, 페이스북이 사용자데이터 수집 기반의 회사로는 최고봉이라고 생각한다 그 반대는 애플인데.. 그래서 요즘 시끌시끌하지만..

> 개인정보 수집을 차단하는 정책을 펼치는 애플과 개인정보를 기반으로 페이스북과 인스타그램이라는 거대한 두 SNS를 이끌어가는 페이스북의 충돌

여튼 기업들은 이용자들의 데이터들을 바탕으로 어떠한 서비스나 상품이 있을시 이와 가장 잘맞을거같은 이용자에게 추천을 해주는식으로 광고를 한다

이러한 다양한 이용자들의 연령, 국가, 성별, 구매목록, 클릭빈도, 관심사 등의 데이터들은 그대로 기업의 운영방향에 이용되며 이용자들의 관심도가 높은 서비스와 상품에는 더욱 투자하고 관심이 저조한 것은 철수를 결정하기도 한다

방문자가 많을수록 좋은 블로그같은 서비스는 지속적으로 방문객을 모으며 고정층을 잡아야한다

고정층을 잡으려면 방문자들이 어떠한 컨텐츠를 가장 많이 클릭하고 가장 많이 머무르는지 알아야하는데 이러한 데이터들을 분석하여 보여주는것이

`Google Analytics` !!

구글 애널리틱스는 본인의 웹사이트에 추적코드를 넣으면 웹사이트에 방문한 사람들이 어떠한 글을 몇명이 보고, 얼마나 머무르고, 그 페이지에서 어디로 이동하는지, 어느나라에서 접속하였는지, 모바일인지 데스크탑인지 등등을 분석하여 차트로 보여준다. 요즘 정말 정말 정말 많이 쓰이는것으로 알고있다! 

내 블로그는 회원가입 없이 불특정다수의 사람들이 접속하는 블로그라서 사용자에대한 정보는 분석 할 수 없으며 성별이나 연령같은것 또한 나오지 않는다

웹사이트에 이용자들의 행동을 분석하는 코드가 들어있다는게 재밌을거같아서 구글서치콘솔을 적용하기 이전부터 이미 적용시켜 놓았었는데 이번에 서치콘솔 적용법을 블로깅하는김에 생각나서 구글 애널리틱스 적용법도 작성하려고한다!

<br>

### Google Analytics

구글 애널리틱스 적용법은 구글서치콘솔보다 훠~~~얼씬 간단하다!

https://www.hedleyonline.com/ko/blog/%EA%B5%AC%EA%B8%80-%EC%95%A0%EB%84%90%EB%A6%AC%ED%8B%B1%EC%8A%A4-ga-%EA%B0%80%EC%9D%B4%EB%93%9C%EB%B6%81/

일단 이 링크를따라들어가 `계정을 생성하고 추적코드를 복사`한다

이후 원래라면 config.js 파일에서 추적코드를 추가해줘야하지만

이또한 gatsby 블로그의 개발자분께서 이미 틀을 잡아놓으셨다

<br>

```javascript
module.exports = {
 title:'코딩코딩',
  ...
 social: {
    twitter: ``,
    github: `Heessong`, 
    medium: ``, 
    facebook: `Heesssong`, 
    linkedin: ``, 
    instagram: `_.heesong._`, 
  },
 ...
 ga:'추적코드 여기에!!!'
}
```

`gatsby-meta-config.js` 파일로가서 `ga` 옆에다가 추적코드를 넣으면 된다..

엄청 간단하다..

혹여나 gatsby블로그가 아니고 다른 블로그를 쓰더라도 결국 추적코드만 삽입하면 되는것이기에 엄청 간단하다!

<br>

```javascript
const metaConfig = require('./gatsby-meta-config')
module.exports ={
	...
      resolve: `gatsby-plugin-google-analytics`,
      options: {
        trackingId: metaConfig.ga,
        head: true,
        anonymize: true,
      },
    },
  ...
}
```

그래도 궁금해서 `config.js` 파일에 들어가봤는데 맨 위 상단에서 `meta-config`를 `require`해주고 있었고

`trackingId`에 떡하니 보이는 `metaConfig.ga`

아하 그래서 코드만 넣어주면 되는거였구만..

이렇게 GA추적코드를 적용하고 구글애널리틱스 사이트로 다시 들어가보면

방금 적용했으면 아무도 안왔을거라 방문자가 0일것이다
